# Backend

Agentes especializados em desenvolvimento de APIs, serviços, bancos de dados e lógica de negócio.

## Propósito

Os agentes backend ajudam a:
- Criar APIs RESTful e GraphQL
- Implementar lógica de negócio
- Otimizar queries de banco de dados
- Gerenciar autenticação e autorização
- Implementar integrações com serviços externos

## Agentes Disponíveis

*Adicione seus agentes aqui conforme forem criados*

## Casos de Uso Comuns

- Desenvolvimento de microserviços
- Criação de APIs
- Otimização de performance
- Implementação de webhooks
- Processamento de jobs assíncronos

## Combinações Recomendadas

- **Backend + DevOps**: Para deploy e escalabilidade
- **Backend + Testadores**: Para testes de integração
- **Backend + Pesquisadores**: Para otimizações
