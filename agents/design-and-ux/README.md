# Design & UX

Agentes especializados em design de interfaces, experiência do usuário, design systems e conteúdo.

## Propósito

Os agentes de Design & UX ajudam a:
- Criar e manter design systems escaláveis e consistentes
- Otimizar experiências de usuário e fluxos
- Polir interfaces com atenção aos detalhes
- Criar interfaces intuitivas e acessíveis
- Desenvolver conteúdo efetivo e coerente
- Implementar princípios de design moderno

## Agentes Disponíveis

### [Design System Builder](./design-system-builder.md)
Especializado em criar e manter design systems escaláveis. Use para definir tokens, componentes reutilizáveis, guidelines de design e documentação do sistema.

### [UX Optimizer](./ux-optimizer.md)
Especializado em otimização de experiência do usuário e fluxos. Use para melhorar usabilidade, resolver problemas de conversão, mapear jornadas e implementar heurísticas de design.

### [UI Polisher](./ui-polisher.md)
Especializado em refinar e polir interfaces de usuário. Use para melhorar layouts, aperfeiçoar componentes, garantir responsividade e elevar a qualidade visual geral.

### [Content Writer](./content-writer.md)
Especializado em criação de conteúdo efetivo para interfaces. Use para escrever microcópias, labels, placeholders, mensagens de erro e documentação de produtos.

### [UI Designer](./ui-designer.md)
Especializado em design de interfaces de usuário e componentes visuais. Use para criar layouts responsivos, definir hierarquia visual e especificações de UI.

### [UX Specialist](./ux-specialist.md)
Especializado em experiência do usuário e usabilidade. Use para otimizar fluxos, resolver problemas de usabilidade e melhorar conversões.

## Casos de Uso Comuns

- Criação de design systems do zero
- Implementação de novos componentes UI
- Redesign de interfaces existentes
- Otimização de fluxos de usuário
- Auditorias de acessibilidade
- Criação de documentação visual
- Pesquisa e testes de usabilidade
- Melhoria de conversão e engajamento

## Combinações Recomendadas

- **Design & UX + Frontend**: Para implementação precisa de designs
- **Design & UX + Testadores**: Para testes de usabilidade e A/B testing
- **Design & UX + Estrategistas**: Para definir direção visual e de produto
- **Design & UX + Pesquisadores**: Para análise de código e componentes existentes
