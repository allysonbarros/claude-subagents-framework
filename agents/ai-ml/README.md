# IA e Machine Learning

Agentes especializados em desenvolvimento de sistemas de inteligência artificial, machine learning e processamento de linguagem natural.

## Propósito

Os agentes de IA e Machine Learning ajudam a:
- Implementar modelos de machine learning e deep learning
- Desenvolver sistemas de IA generativa
- Criar pipelines de processamento de dados
- Integrar LLMs e RAG systems
- Construir chatbots e agentes autônomos
- Otimizar e fazer fine-tuning de modelos

## Agentes Disponíveis

### [AI Agent Architect](./ai-agent-architect.md)
Especializado em design e implementação de agentes de IA autônomos. Use para arquitetar sistemas multi-agentes, implementar loops de reasoning e coordenar agentes complexos.

### [Claude Agent SDK Specialist](./claude-agent-sdk-specialist.md)
Especializado no Claude Agent SDK para construir agentes que podem usar tools. Use para criar agentes que integram com APIs, executam ações complexas e interagem com sistemas externos.

### [CrewAI Specialist](./crewai-specialist.md)
Especializado em CrewAI para orquestração de agentes colaborativos. Use para criar crews de agentes especializados com diferentes papéis e capacidades.

### [LangChain Specialist](./langchain-specialist.md)
Especializado em LangChain para construir aplicações com LLMs. Use para criar chains, memories, tools e integrar com múltiplos LLMs.

### [OpenAI Swarm Specialist](./openai-swarm-specialist.md)
Especializado em OpenAI Swarm framework para coordenação de agentes. Use para criar sistemas de agentes lightweight com hand-offs dinâmicos.

### [Prompt Engineering Specialist](./prompt-engineering-specialist.md)
Especializado em engenharia de prompts e otimização de performance de modelos. Use para craftar prompts efetivos, implementar chain-of-thought e otimizar outputs de LLMs.

### [RAG Specialist](./rag-specialist.md)
Especializado em Retrieval-Augmented Generation. Use para construir sistemas que combinam busca semântica com geração de texto, criar knowledge bases e implementar citações.

### [Gradio Specialist](./gradio-specialist.md)
Especializado em criar interfaces web com Gradio para demonstração de modelos. Use para prototipar rapidamente aplicações de IA e criar demos interativas.

### [Streamlit Specialist](./streamlit-specialist.md)
Especializado em Streamlit para construir aplicações de dados e IA. Use para criar dashboards, demos de modelos e aplicações interativas com Python puro.

## Casos de Uso Comuns

- Desenvolvimento de LLM applications
- Criação de agentes autônomos
- Implementação de RAG systems
- Fine-tuning de modelos
- Análise de sentimento e NLP
- Chatbots e assistentes virtuais
- Sistemas de recomendação com IA
- Processamento e análise de documentos

## Combinações Recomendadas

- **IA/ML + Pesquisadores**: Para análise de modelos e avaliação de alternativas
- **IA/ML + Backend**: Para integração de modelos em APIs e serviços
- **IA/ML + Data Engineering**: Para pipelines de dados e ETL para treinamento
- **IA/ML + Devops**: Para deployment e scaling de modelos
