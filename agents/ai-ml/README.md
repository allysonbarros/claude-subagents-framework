# AI/ML

Agentes especializados em inteligência artificial, machine learning, e desenvolvimento de aplicações com LLMs.

## Propósito

Os agentes AI/ML ajudam a:
- Desenvolver aplicações com Large Language Models (LLMs)
- Criar sistemas multi-agent
- Implementar RAG (Retrieval Augmented Generation)
- Construir interfaces para modelos de ML
- Otimizar prompts e workflows de IA
- Arquitetar soluções de IA robustas e escaláveis

## Agentes Disponíveis

### [LangChain/LangGraph Specialist](./langchain-specialist.md)
Especializado em aplicações com LLMs usando LangChain e LangGraph. Use para criar agents, chains, RAG systems e workflows complexos.

### [Gradio Specialist](./gradio-specialist.md)
Especializado em interfaces ML com Gradio. Use para criar demos rápidos, compartilhar modelos e interfaces interativas.

### [Streamlit Specialist](./streamlit-specialist.md)
Especializado em data apps e dashboards com Streamlit. Use para visualização de dados e ML apps interativos.

### [CrewAI Specialist](./crewai-specialist.md)
Especializado em sistemas multi-agent com CrewAI framework. Use para orquestrar agentes colaborativos com roles e tasks.

### [Claude Agent SDK Specialist](./claude-agent-sdk-specialist.md)
Especializado em construir agents com Anthropic's Agent SDK e Model Context Protocol. Use para criar Claude-powered agents com MCP e tools.

### [OpenAI Swarm Specialist](./openai-swarm-specialist.md)
Especializado em sistemas multi-agent com OpenAI Swarm framework. Use para coordenar agents com handoffs e routines.

### [RAG Specialist](./rag-specialist.md)
Especializado em Retrieval Augmented Generation systems. Use para indexação, retrieval e geração com contexto.

### [Prompt Engineering Specialist](./prompt-engineering-specialist.md)
Especializado em otimização, teste e avaliação de prompts para LLMs. Use para criar, testar e avaliar prompts efetivos.

### [AI Agent Architect](./ai-agent-architect.md)
Especializado em arquitetura de sistemas multi-agent e workflows. Use para projetar e arquitetar sistemas complexos de IA.

## Casos de Uso Comuns

- Desenvolvimento de chatbots e assistentes virtuais
- Sistemas de recomendação inteligentes
- Aplicações RAG para documentação
- Automação com multi-agents
- Análise e processamento de linguagem natural
- Demos e protótipos de ML

## Combinações Recomendadas

- **AI/ML + Backend**: Para APIs de ML
- **AI/ML + Data Engineering**: Para pipelines de dados
- **AI/ML + Frontend**: Para interfaces de IA
- **AI/ML + DevOps**: Para deploy de modelos
