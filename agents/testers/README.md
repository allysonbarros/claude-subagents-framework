# Testadores

Agentes especializados em testes automatizados, garantia de qualidade e estratégias de testing.

## Propósito

Os agentes testadores ajudam a:
- Criar testes unitários e de integração
- Implementar testes E2E
- Desenvolver estratégias de testing
- Identificar edge cases
- Melhorar cobertura de testes

## Agentes Disponíveis

*Adicione seus agentes aqui conforme forem criados*

## Casos de Uso Comuns

- Aumento de cobertura de testes
- Testes de regressão
- Testes de performance
- Testes de acessibilidade
- Configuração de CI/CD

## Combinações Recomendadas

- **Testadores + Frontend**: Para testes de componentes
- **Testadores + Backend**: Para testes de API
- **Testadores + DevOps**: Para testes em pipeline
