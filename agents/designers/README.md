# Designers

Agentes especializados em design de interface, experiência do usuário e design systems.

## Propósito

Os agentes designers ajudam a:
- Criar interfaces intuitivas e acessíveis
- Desenvolver design systems consistentes
- Implementar princípios de UX
- Criar protótipos e mockups
- Garantir responsividade

## Agentes Disponíveis

### [UI Designer](./ui-designer.md)
Especializado em design de interfaces de usuário e componentes visuais. Use para criar layouts responsivos, definir hierarquia visual e especificações de UI.

### [UX Specialist](./ux-specialist.md)
Especializado em experiência do usuário e usabilidade. Use para otimizar fluxos, resolver problemas de usabilidade e melhorar conversões.

### [Design System Builder](./design-system-builder.md)
Especializado em criar e manter design systems escaláveis. Use para definir tokens, componentes reutilizáveis e guidelines de design.

## Casos de Uso Comuns

- Criação de novos componentes UI
- Redesign de interfaces existentes
- Implementação de design systems
- Otimização de fluxos de usuário
- Auditorias de acessibilidade

## Combinações Recomendadas

- **Designers + Frontend**: Para implementação precisa de designs
- **Designers + Testadores**: Para testes de usabilidade
- **Designers + Estrategistas**: Para definir direção visual
