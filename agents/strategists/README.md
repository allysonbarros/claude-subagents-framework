# Estrategistas

Agentes especializados em planejamento estratégico, arquitetura de sistemas e tomada de decisões técnicas de alto nível.

## Propósito

Os agentes estrategistas ajudam a:
- Definir arquiteturas e padrões de projeto
- Planejar roadmaps técnicos
- Avaliar trade-offs de decisões técnicas
- Criar especificações de alto nível
- Definir requisitos e prioridades

## Agentes Disponíveis

*Adicione seus agentes aqui conforme forem criados*

## Casos de Uso Comuns

- Início de novos projetos
- Refatorações grandes
- Definição de padrões de código
- Planejamento de sprints técnicas
- Avaliação de tecnologias

## Combinações Recomendadas

- **Estrategistas + Pesquisadores**: Para decisões baseadas em análise técnica profunda
- **Estrategistas + Testadores**: Para garantir que a arquitetura seja testável
- **Estrategistas + DevOps**: Para arquiteturas cloud-native
