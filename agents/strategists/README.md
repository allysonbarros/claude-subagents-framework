# Estrategistas

Agentes especializados em planejamento estratégico, arquitetura de sistemas e tomada de decisões técnicas de alto nível.

## Propósito

Os agentes estrategistas ajudam a:
- Definir arquiteturas e padrões de projeto
- Planejar roadmaps técnicos
- Avaliar trade-offs de decisões técnicas
- Criar especificações de alto nível
- Definir requisitos e prioridades

## Agentes Disponíveis

### [Product Manager](./product-manager.md)
Especializado em análise de produto, definição de requisitos e priorização de features. Use para transformar ideias em especificações claras e criar roadmaps.

### [Tech Architect](./tech-architect.md)
Especializado em arquitetura de sistemas e decisões técnicas de alto nível. Use para projetar sistemas escaláveis e definir padrões arquiteturais.

### [API Designer](./api-designer.md)
Especializado em design de APIs RESTful e GraphQL. Use para criar especificações de API consistentes e bem documentadas.

### [System Architect](./system-architect.md)
Especializado em projetar arquiteturas de sistemas escaláveis e resilientes. Use para design arquitetural, diagramas C4, padrões de escalabilidade e disaster recovery.

## Casos de Uso Comuns

- Início de novos projetos
- Refatorações grandes
- Definição de padrões de código
- Planejamento de sprints técnicas
- Avaliação de tecnologias

## Combinações Recomendadas

- **Estrategistas + Pesquisadores**: Para decisões baseadas em análise técnica profunda
- **Estrategistas + Testadores**: Para garantir que a arquitetura seja testável
- **Estrategistas + DevOps**: Para arquiteturas cloud-native
