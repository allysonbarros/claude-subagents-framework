# Pesquisadores

Agentes especializados em exploração de código, pesquisa técnica e análise de sistemas existentes.

## Propósito

Os agentes pesquisadores ajudam a:
- Explorar e entender codebases grandes
- Pesquisar soluções técnicas
- Analisar dependências e integrações
- Documentar sistemas existentes
- Encontrar padrões e anti-padrões no código

## Agentes Disponíveis

*Adicione seus agentes aqui conforme forem criados*

## Casos de Uso Comuns

- Onboarding em projetos existentes
- Debug de problemas complexos
- Auditorias de código
- Migração de sistemas legados
- Análise de performance

## Combinações Recomendadas

- **Pesquisadores + Estrategistas**: Para refatorações informadas
- **Pesquisadores + Testadores**: Para identificar áreas não testadas
- **Pesquisadores + Backend**: Para otimização de APIs
