# Frontend

Agentes especializados em desenvolvimento de interfaces web, frameworks JavaScript e experiências do usuário.

## Propósito

Os agentes frontend ajudam a:
- Implementar componentes reutilizáveis
- Gerenciar estado da aplicação
- Otimizar performance do frontend
- Integrar com APIs
- Implementar animações e interações

## Agentes Disponíveis

*Adicione seus agentes aqui conforme forem criados*

## Casos de Uso Comuns

- Desenvolvimento de SPAs
- Criação de dashboards
- Implementação de formulários complexos
- Integração com WebSockets
- Progressive Web Apps

## Combinações Recomendadas

- **Frontend + Designers**: Para fidelidade visual perfeita
- **Frontend + Backend**: Para integração full-stack
- **Frontend + Testadores**: Para testes E2E
