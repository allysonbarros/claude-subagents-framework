# Frontend

Agentes especializados em desenvolvimento de interfaces web, frameworks JavaScript e experiências do usuário.

## Propósito

Os agentes frontend ajudam a:
- Implementar componentes reutilizáveis
- Gerenciar estado da aplicação
- Otimizar performance do frontend
- Integrar com APIs
- Implementar animações e interações

## Agentes Disponíveis

### [React Specialist](./react-specialist.md)
Especializado em desenvolvimento React moderno com hooks e boas práticas. Use para criar componentes, implementar hooks customizados e seguir patterns do React.

### [State Manager](./state-manager.md)
Especializado em gerenciamento de estado com Redux, Zustand, Context API. Use para arquitetar estado global, otimizar performance e escolher soluções adequadas.

### [Performance Optimizer](./performance-optimizer.md)
Especializado em otimização de performance e Core Web Vitals. Use para reduzir bundle size, otimizar re-renders e melhorar métricas de loading.

### [Performance Engineer](./performance-engineer.md)
Especializado em Web Vitals, bundle optimization e análise de performance. Use para LCP/FID/CLS, code splitting, caching e monitoramento contínuo.

### [Mobile Optimizer](./mobile-optimizer.md)
Especializado em otimização para dispositivos móveis e responsive design. Use para touch interfaces, viewport configuration, performance em mobile e PWAs.

## Casos de Uso Comuns

- Desenvolvimento de SPAs
- Criação de dashboards
- Implementação de formulários complexos
- Integração com WebSockets
- Progressive Web Apps

## Combinações Recomendadas

- **Frontend + Designers**: Para fidelidade visual perfeita
- **Frontend + Backend**: Para integração full-stack
- **Frontend + Testadores**: Para testes E2E
