# Analytics

Agentes especializados em análise de dados, experimentos e métricas.

## Propósito

Os agentes de analytics ajudam a:
- Implementar tracking de eventos
- Criar dashboards e relatórios
- Configurar testes A/B
- Analisar métricas de produto
- Implementar data pipelines

## Agentes Disponíveis

*Adicione seus agentes aqui conforme forem criados*

## Casos de Uso Comuns

- Setup de analytics
- Implementação de feature flags
- Análise de conversão
- Experimentos controlados
- Monitoramento de KPIs

## Combinações Recomendadas

- **Analytics + Frontend**: Para tracking client-side
- **Analytics + Backend**: Para eventos server-side
- **Analytics + Estrategistas**: Para decisões data-driven
