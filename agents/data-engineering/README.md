# Data Engineering

Agentes especializados em engenharia de dados, data warehouses e plataformas de analytics.

## Propósito

Os agentes de Data Engineering ajudam a:
- Construir pipelines de dados escaláveis
- Transformar e modelar dados com dbt
- Implementar data warehouses modernos
- Processar big data com Spark
- Criar arquiteturas lakehouse
- Otimizar workflows de analytics

## Agentes Disponíveis

### [DBT Specialist](./dbt-specialist.md)
Especializado em transformação de dados com dbt (data build tool). Use para criar models, tests, documentação de pipelines e workflows de transformação.

### [Snowflake Specialist](./snowflake-specialist.md)
Especializado em Snowflake data warehouse e cloud analytics. Use para queries, schemas, Snowpipe, streams e tasks.

### [Databricks Specialist](./databricks-specialist.md)
Especializado em Databricks lakehouse com Delta Lake e Spark. Use para notebooks, Delta Lake, Spark SQL e ML workflows.

### [Spark Specialist](./spark-specialist.md)
Especializado em processamento big data com Apache Spark. Use para transformações, analytics e ML em escala com PySpark.

## Casos de Uso Comuns

- ETL/ELT pipelines
- Data warehousing
- Analytics engineering
- Big data processing
- Real-time data streaming
- Data quality e testing
- Lakehouse architecture

## Combinações Recomendadas

- **Data Engineering + Analytics**: Para dashboards e relatórios
- **Data Engineering + AI/ML**: Para feature engineering
- **Data Engineering + Backend**: Para APIs de dados
- **Data Engineering + DevOps**: Para orchestração de pipelines
